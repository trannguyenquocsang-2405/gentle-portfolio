# Frontend - Gentle Portfolio

This is a [Next.js](https://nextjs.org) project bootstrapped with `create-next-app` and migrated from Vite.

## Getting Started

First, install dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
